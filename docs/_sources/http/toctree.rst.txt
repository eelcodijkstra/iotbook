Het HTTP-protocol is de basis van het web.
Het wordt ook gebruikt voor het "programmable web":
met behulp van web-APIs kun je (web)toepassingen maken waarin je
allerlei diensten kunt gebruiken die in het web worden aangeboden.
Enkele voorbeelden:

* inbedden ("embedden") van kaarten, video's of documenten in een webpagina;
* automatisch vertalen van de tekst van je website, via Google translate e.d.;
* raadplegen van een database (zoals CouchDB);

In het Internet of Things kom je het HTTP-protocol op de volgende plaatsen tegen:

* voor het configureren van een IoT-knoop via WiFi;
* als web-API voor het aansturen van de actuatoren of uitlezen van de sensorwaarden van een IoT-knoop;
* als web-API van een controller;
* als web-API van een webdienst (cloud-services).
